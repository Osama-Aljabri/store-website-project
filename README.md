# Multi-Category Online Store System

## 📌 Project Overview
This project is an e-commerce system that allows customers to browse products, add items to cart, and complete purchases online.

## 🎯 Objectives
- Improve online shopping experience
- Save time and effort for customers
- Help merchants manage products and data efficiently

## 🧠 Problem Solved
- Customers cannot easily see products on social media
- Wasting time going to physical stores
- Merchants need organized customer data

## ⚙️ Features
- Product browsing and filtering
- Shopping cart system
- Order management
- Customer and merchant accounts

## 🏗️ Project Components
- Work Breakdown Structure (WBS)
- Network Diagram
- Project Schedule (MS Project)
